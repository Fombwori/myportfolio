PROJECT NAME:
Faith Ombwori – Personal Portfolio Website

DESCRIPTION:
This project is a personal and professional portfolio website for
Faith Kwamboka Ombwori, a Cybersecurity Expert and Women in Tech Leader.
The website showcases Faith’s background, achievements, education,
professional expertise, and contributions in cybersecurity and technology
leadership across Africa.

FEATURES:

Responsive web design for all devices.

Sections for About, Education, Skills, Projects, and Contact.

Custom HTML, CSS, and JavaScript design.

Integration with social media and professional links.

Hosted using GitHub Pages.

PROJECT STRUCTURE:
myportfolio/
│
├── index.html          → Main homepage
├── about.html          → About section
├── projects.html       → Projects showcase
├── contact.html        → Contact form or details
│
├── css/
│   └── style.css       → Main stylesheet
│
├── js/
│   └── script.js       → JavaScript functions
│
├── images/
│   └── (profile, banners, icons)
│
└── README.txt          → Project information

TECHNOLOGIES USED:

HTML5

CSS3

JavaScript

GitHub Pages (Hosting)

SETUP AND INSTALLATION:

Clone the repository 
git clone https://github.com/fombwori/myportfolio.git
Open the folder
cd myportfolio
Launch the site locally

Open index.html in a web browser
OR
Use a local server (e.g., VS Code Live Server)

DEPLOYMENT (GitHub Pages):

Push all project files to the GitHub repository.

Go to repository Settings → Pages.

Under Source, select the main branch and / (root) folder.

Click Save.

The site will be published at:
https://fombwori.github.io/myportfolio/

USAGE:
Navigate to each section via the menu.
View professional background, achievements, and contact details.
